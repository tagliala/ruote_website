ruote users

A non-exhaustive list of organizations/companies using ruote.

Some of these are service integrators, don’t hesitate to contact them if you need support / advice. If you’d like to be mentioned, please say so on the mailing list or on IRC (freenode.net #ruote).

  • the Fermilab is using ruote (USA)

  • rackspace (USA) and ThoughtWorks are orchestrating data center processes with ruote
  • onit (USA) use ruote in their light-weight tool for tracking projects, deals, transactions, cases and litigation
  • howcast (USA)
  • crowdflower (USA)
  • maestrodev (USA)

asia

  • Cookpad (Japan)

  • Planquadrat Software-Integration (Germany)

  • xaop (Belgium) (check out their open source projects)
  • OrganisedMinds are using ruote (Germany)
